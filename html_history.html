CSC104 HTML and CSS History

HTML Explained

HTML stands for HyperText Markup Language. HTML provides the structure of the webpage by marking up the text. You can (and should) use variety of HTML tags to build the webpage content. HTML does not have many rules (other than if you open a tag, you should usually close it with a matching tag). One and only h1 tag is a best practice.

History of HTML

  • HTML 1.0 - developed in the early 1990's
  • HTML 2.0 - released in 1995
  • HTML 3.2 - released in 1997
  • HTML 4.01 - released in 1999
  • HTML 5 - release in 2014 (current version of HTML) - living standard

CSS Explained

CSS stands for Cascading Style Sheets. CSS is a language that exists to make HTML look better. CSS can control layout, colors, fonts - basically any visual element on the page can be changed from the default browswer style using CSS.

History of CSS

  • CSS1 released in 1996
  • CSS2 released in 1998
  • CSS3 released in 2011 (current version of CSS) - living standard

CSS Structure

        /* HTML element */
        selector{
            property: value;
        }

        /* custom class */
        .redText{
            color: red;
        }

        /* class based on id of the HTML element */
        #h2CSSstructure{
            color: blue;
        }
  • CSS selectors: they can be an HTML element, custom class, based on the id of the HTML element
  • CSS helps with the separation of content from style
  • Helps with maintainability
  • Accessibility issues can be more easily managed

3 Ways to Include CSS

  • External: CSS is in a separate CSS file (best practice)
  • Embedded: CSS on a single page
  • Inline: CSS is set on a specific HTML element (use with caution)
